# python-experiment
Python 数据分析与文本挖掘实践项目
本仓库是一个综合性的Python学习与实践项目，涵盖了从基础语法到专业数据分析库的多个实验。内容分为两大主线：一是基于NumPy、Matplotlib和Pandas的数据分析基础与可视化；二是基于jieba中文分词库的经典文学作品《红楼梦》的文本挖掘。

📚 项目内容
本仓库包含了四个独立的实验报告及其对应的代码实现：

实验一：Python字符串程序设计

实现一个密码强度检查器，评估字符串的安全性。

核心项目：使用jieba库对《红楼梦》全文进行分词，统计并分析小说中主要人物的出场次数，最终生成排行榜。

实验二：基于NumPy的数据分析

生成并可视化服从正态分布的随机数据。

实现基础的矩阵运算（加法、减法、乘法）。

使用np.loadtxt读取外部数据文件并进行格式转换。

实验三：基于Matplotlib库的数据可视化

绘制沪指综合指数收盘数据的趋势图。

分析人均消费与人均国民收入的关系，并绘制散点图。

绘制学生考试成绩的频率直方图，并进行统计分析。

实验四：基于Pandas的数据分析

模拟从MySQL数据库读取股票数据，计算移动平均线并绘制技术分析图表。

使用Pandas读取Excel文件，进行数据排序与筛选。

🛠️ 技术栈
核心语言: Python 3

数据分析: NumPy, Pandas

数据可视化: Matplotlib, Seaborn

文本挖掘: Jieba

数据库交互: SQLAlchemy (模拟)

文件处理: Excel (.xls, .xlsx), CSV, TXT

✨ 特点
理论与实践结合：每个实验都包含明确的目的、步骤、代码和结果分析。

经典案例：从金融数据分析到古典文学名著《红楼梦》的挖掘，案例生动有趣。

完整的项目结构：包含了数据集（如《红楼梦》全文）、配置文件、主程序脚本和详细的实验报告。

适合初学者：代码注释详细，逻辑清晰，是学习Python数据分析生态的绝佳入门资料。

🚀 如何开始
克隆本仓库到本地。

确保已安装 Python 3.x。

安装所需依赖库：

bash
pip install numpy pandas matplotlib jieba scipy sqlalchemy openpyxl
直接运行各个实验的主文件（如 analyze.py, experiment2.py, experiment3_main.py）查看效果。
